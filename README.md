# 🎓 广东省高考志愿填报参考系统

专为广东省高考生打造的志愿填报辅助参考平台，基于历年录取数据与智能算法，提供科学合理的填报建议。

## ✨ 核心功能

- 📝 **分数排名输入**：录入高考分数、全省排名、选科组合
- 🎯 **智能推荐院校**：基于历年数据与**年份难度对标**，推荐冲稳保三档院校
- 📋 **模拟志愿填报**：按广东省 45 个志愿格式，模拟正式填报
- 📊 **数据可视化**：排名趋势、录取概率、年份难度对比
- 🏫 **院校对比分析**：多院校横向对比

## 🔧 技术栈

- **前端框架**：React 18 + TypeScript
- **UI 组件**：Ant Design 5
- **图表库**：ECharts
- **状态管理**：Zustand
- **构建工具**：Vite
- **路由**：React Router v6

## 🚀 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:5173
```

## 📦 构建生产版本

```bash
npm run build
npm run preview  # 预览生产构建
```

## ⚠️ 免责声明

本系统提供的所有推荐结果及录取概率均基于历史数据推算，**仅供参考**，不构成正式填报建议。请以广东省教育考试院及院校官方公布的招生信息为准。

## 📊 数据来源

- 广东省教育考试院官网
- 各院校招生网
- 阳光高考网

数据范围为 2022-2025 年广东省高考录取数据。
